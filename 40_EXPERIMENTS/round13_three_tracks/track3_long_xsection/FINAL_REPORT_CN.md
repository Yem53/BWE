# Track 3 — 截面排名做多 最终报告（中文白话）

> 一句话结论：**做多侧仍然没有稳健 alpha。** 换了三个全新角度（全市场截面排名 /
> 多特征组合 / 强平背景）独立搜索，全部在 dev 上不达门槛、holdout 再次确认，
> 与 round12 的"波动暴增→做多"穷举证伪**完全吻合**。妖币市场做多是结构性负期望，
> 任何"谁最猛就追多"的信号都被 0.15% 往返成本吃掉，挑出来的少数赢家都是肥尾幸存者偏差。

- 生成时间：2026-05-29
- 数据窗口：K线归档 640 币 × 1h，2026-01-27 → 2026-05-25（~119 天）
- **holdout 封存线：2026-04-24 00:00 UTC（now−35d）**；dev = 01-27 → 04-24（~87 天，4 折 walk-forward）；holdout 只在最后读 1 次
- 成本模型：往返 taker **0.15%**（COST_RT，与 round8/round12 对齐）；急拉成交风险用入场bar overshoot >0.8% 标注
- 共试 **约 188 个**做多配置/条件测试（A 主扫描 112 + A2 诊断 21 + A3 审查 + B 嵌套CV + C 探索 14 + D holdout 16），dev 上 **0 个**达"正edge + 4/4折一致 + 单调"门槛

---

## 0. 与已证伪方向的区别（没有重复造轮子）

round12 已穷举证伪 per-coin 时序"波动暴增→做多"（648网格+9细筛+18 onset 全军覆没）。
本轮**刻意避开那条路**，换互补角度。最关键的区别：round12 报告自己写明它的"截面"
其实是 **"同一时刻触发事件集合内"的排名**（axis7，24测试0过），**不是全市场640币完整截面**。
本 track A 做的正是真正未探索的 **全市场每小时横截面排名**（BWE 扫描器式"谁此刻最猛"）。

---

## 1. Track A（重点）— 全市场截面排名做多

**方法**：每小时 t 对所有活跃币（~620个）按相对强度排名（6种信号：mom_3h/6h/12h/24h、
量能z48、accel），做多最强的 top-K（K=3/5/10/20），t+1 开盘入场、持 H 小时（1/2/4/6）、
收盘平、扣成本。严格无前视（排名用 close[t]，入场在 t+1，已单元测试核对）。判定门槛：
top-K 净收益明显超"裸做多全市场"基线 + 4 折一致 + 强弱分层单调 + holdout 确认。

### 结果（dev）
| 项 | 数字 | 含义 |
|---|---|---|
| 裸做多全市场基线 H1-H6 | net **−0.15%~−0.16%**，胜率 **34-37.5%** | 做多本身结构性负期望 |
| 截面 top-K 做多最好 edge | accel H2 K3 edge=+0.13%，但 top-K 自身仍 **−0.027%**、t=**−0.24** | 只是"比裸做多少亏"，绝对值仍亏、完全不显著 |
| **达门槛候选数** | **0 / 112** | 正edge+4/4折+单调 一个都没有 |

### 截面信号到底有没有方向性信息？（gross 诊断，A2）
- **短期动量（mom_3h/6h）有微弱"延续"**：long-short spread = +0.03%~+0.05%，t≈2.1-2.7（弱显著）。
- **长期动量（mom_24h）反转**：spread = −0.02%~−0.04%，t≈−1.7~−2.2（涨24h的反而要跌 → 支持做空）。
- **致命点**：即使最强10%币的**毛收益**也只有 +0.01%~+0.024%，**扣 0.15% 成本后全变成 −0.13%~−0.18%**。
  这是教科书式的"信号存在但被交易成本吃掉"，在做多侧无法变现。

### 条件化也救不回来（A2）
- **BTC regime**：bear 时 mom_3h H4 做多 net=+0.015%（t+5.5）——但 +0.015% ≈ 打平，t 高只因样本大（n3万），经济意义为 0；bull/chop 都负。
- **时段**：BH-FDR(α=0.10) 后 **只有 h17 一个小时显著为正**，却有 **16 个小时显著为负**（亚洲时段 h0-h11 几乎全负，最差 h9 = −0.31% t−4.63）。一边倒指向"做空最强币才对"。

### 唯一"正"候选 h17 被审查证明是过拟合（A3）
对"UTC 14-18 截面做多 top-10%"做反过拟合三连审查：
- **跨折不一致**：4 折只有 2 折正（fold1/fold2 都负）；
- **97% 收益来自 3 天**（02-28 / 04-16 / 02-07），正收益天数仅 38/87（44%）；
- **58% 收益来自 3 个币**（SIRENUSDT/RAVEUSDT/AKEUSDT）。
→ 判定 **KILLED**：是少数妖币单日暴拉被截面恰好抓到的肥尾，不是可交易的稳定时段 alpha。

**Track A 结论：全市场截面排名做多无正期望 alpha。最强势币的截面动量延续效应真实存在但 <0.05%，远小于成本，做多任何分位都亏钱。**

## 2. Track B — 多特征组合（嵌套CV，必须打赢最好单特征）

**方法**：截面框架下，每币每小时取 9 个价量截面特征（多周期动量、accel、波动率、量能、
24h 位置、本bar body），全部 **横截面 rank 到 [0,1]**（无前视、scale-free）。目标=做多
未来 4h 净收益 / 是否跑赢截面中位数。**嵌套 CV**：外层 4 折时间切分，内层在训练区再切时间
选 L2（手写 logistic 回归，已单元测试）。判据：组合的 top-decile 净收益必须打赢"最好单特征"
的 top-decile，且 4 折一致为正。

### 结果（dev）
| 项 | 数字 |
|---|---|
| 组合 top-decile 净收益（4折均值） | **−0.137%** |
| 最好单特征 同口径 | −0.165% |
| 组合打赢单特征？ | **是**（4/4 折都略胜，组合确实榨出一点额外信息） |
| 组合 4 折全正？ | **否**（3/4 折为负：−0.37% / −0.13% / −0.16%，只有含 04-16 暴涨日的 fold3 +0.11%） |

**Track B 结论：组合虽然打赢了单特征，但打赢的是"谁亏得少"——组合本身净收益 −0.137%（亏钱）且跨折不稳（3/4负）。按纪律 REJECT，不进 holdout。** 再次印证做多侧的特征单个或组合都跨不过成本。fold3 转正又是那个被 A3 证明的"少数妖币暴拉日"效应。

## 3. Track C — 强平背景做多（逼空燃料）【探索性观察，不可验证】

> ⚠️ **硬约束**：自采强平数据只覆盖 2026-05-03 → 05-29，**100% 落在封存的 holdout 里**
> （cut 04-24），dev 上完全无法筛。以下是 **holdout 内的探索性观察，不是验证结论，不可上实盘**。
> 且强平与拉升**共生**（pump 同时推价 + 触发空头强平），即使正向也可能是动量残留而非强平因果。

**方法**：空头强平 = BUY side 强平（强制买入平空）。按币按小时聚合空头强平名义额，
严格用 t 时刻强平 → 做多 t+1（不与同期收益相关，避免共生幻觉），看 top-decile 强平币的前瞻收益。

### 观察（holdout，~22天重叠，20.7万条空头强平）
| hold | top-decile 强平做多 | 市场基线做多 | 胜率 |
|---|---|---|---|
| H1 | −0.107% | −0.149% | 39.1% vs 32.8% |
| H2 | −0.112% | −0.149% | 41.6% vs 34.7% |
| H4 | −0.133% | −0.146% | 41.8% vs 36.0% |
| H6 | −0.096% | −0.142% | 44.0% vs 36.8% |
| **强平加速 H2** | **−0.04%**（最接近0） | — | 44.3% |

**Track C 观察：空头强平多的币接下来确实跌得少一点、胜率高一点（squeeze fuel 有一丝痕迹），但没有任何配置变成正期望，幅度不足以扣过成本做多。** 即使这点微弱正向也可能是同期动量残留。**明确标注：holdout 数据、不可当验证、不可上实盘。** 价值仅在于：未来攒够 dev 期的强平历史后，"强平加速"值得专门开一个 dev 窗口再验。

## 4. Track D — Holdout 最终确认（读 1 次）

dev 上 A/B **0 个做多候选达标**，所以严格说无候选可"reveal"。这里做的是**确认负结论**的
单次 holdout pass（不是择优后验证，避免过拟合 holdout）。

### 结果（holdout 2026-04-24 → 05-25，768 bars）
| 配置 | 净收益 | t | 说明 |
|---|---|---|---|
| 裸做多 H4 | **−0.131%**（wr 35.9%） | — | 确认做多本质负期望 |
| 截面 top-K 做多 mom_3h/6h（K5/10/20，H2/H4） | **全部 −0.13%~−0.30%** | 多数显著为负（K20 t=−4~−6） | 做多最强币在 holdout **显著比裸做多还亏**（追高被套重演） |
| **过拟合的 dev 选项 h17** | dev +0.186% → **holdout −0.195%（t−1.97）** | — | 完美演示 dev 挑出的"信号"无法泛化 |
| h14-18 整段 | +0.031%（t=0.67） | 不显著 | 噪声（dev 早已跨折不一致） |

**Track D 结论：holdout 确认做多侧无 edge——每个截面做多配置（含过拟合 dev 选项）在 holdout 上都净负或不显著为正。**

## 5. 反过拟合 / 前视 / 成交性 自查

- **前视**：排名用 close[t]，执行用 open[t+1]，已用合成数据单元测试逐笔核对对齐（hold=1/2 都对、末尾无未来bar的格子正确剔除为NaN）。截面 rank 在同一时刻横截面计算，天然无前视（不涉及未来时间戳）——这正是截面 rank 比"per-coin 历史百分位"安全的地方。
- **多重比较**：共试约 188 配置/条件。24 小时分桶做了 **BH-FDR(α=0.10)**；任何"显著"项都额外查跨折一致 + 集中度，**没有单点显著就当真**。
- **幸存者偏差**：唯一像信号的 h17/14-18 被证明 97% 收益来自 3 天、58% 来自 3 币 → 肥尾，不可交易。这与任务先验"GUA(+23%) 是幸存者偏差"的截面版完全一致。
- **成交性**：截面做多追的恰恰是"全市场此刻最猛"的币，正是急拉、滑点最大、最可能成交不了的标的（>0.8% overshoot 会弃单），所以**即便毛收益微正，实盘还要再打折**——更不可能变现。
- **强平不可验证**：Track C 数据 100% 在 holdout，已全程明确标注。

## 6. 最终交付结论

| 交付项 | 结论 |
|---|---|
| ① 截面排名做多 dev/holdout | **0 候选达标**；dev 最强 edge 仍亏、不显著、单点走运；holdout 截面做多显著比裸做多还亏 |
| ② 多特征组合 | 组合打赢单特征但**自身净负、3/4折负** → REJECT，不进 holdout |
| ③ 强平做多探索 | squeeze fuel 有一丝痕迹（跌得少/胜率高）但仍净负；**holdout数据、不可验证、不可上实盘** |
| ④ holdout 确认的做多规则 | **无。诚实报负：没有稳健做多 alpha，与 round12 穷举证伪一致。** |
| ⑤ 共试组合 + FDR + 顾虑 | ~188 配置；BH-FDR(0.10)；前视已单测核对；成交性顾虑（追最猛币=最难成交）记录在案 |

**给项目的话**：本轮从三个全新互补角度（截面/组合/强平）独立验证了"妖币做多无稳健 alpha"
这一强先验，证据链完整（dev 4折 + FDR + 集中度审查 + holdout 确认）。**有方向性信息的反而是
做空侧**——截面长动量反转、16 个时段做多显著为负、做空最强势币会盈利，都与项目既有的
"妖币 SHORT-only"结论相互印证。建议：做多侧资源可以收手；若未来要再碰做多，只有"强平加速"
这一个未被 dev 验过的角度值得在攒够强平历史后专门开窗口试，其余角度已基本穷尽。
