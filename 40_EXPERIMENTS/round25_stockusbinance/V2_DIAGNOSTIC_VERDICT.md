# stockusbinance v2 三项诊断 — 整合判决(2026-06-13)

用户"全部都做吧": carry核算 / β对冲重算 / 费率倾斜。三项全部回来, 结论是 **重大的 Rule #0 时刻 —— 数据动摇了 stockusbinance 的核心前提**。零真钱, 发生在花钱之前。

## 三项结果

### 任务1 双腿carry → ☠️ $1000不值得
- 11标的真实结算: MRVL近期36%/MU 20%/NOK崩到3%。但 **11/11 标的"资金费收入/基差波动 < 1"**:
  单日基差随机偏离(3-10bp)是单次8h结算收入(0.4-4bp)的2-14倍 → 建仓时点基差不利, 一天就吞掉数周收益。
- 高费率衰减最快(NOK 45%→3%, 是新上市投机退潮)。$500腿月净个位数, AMZN/TSM已转负。
- 唯一保留: 已持正股的 MRVL/MU 可微试, 不推荐。

### 任务2 β对冲重算 → 🔴 核心前提被动摇(最重要发现)
- **Buy侧 −1.88%**: ~70%是大盘β(信号期QQQ −1.62%), 对冲后 −0.25~−0.44% **仍负** → 无多头alpha。
- **Sell侧 +1.30%(原以为的edge)**: 对冲后 −0.18~−1.62% **转负/归零** → 那段QQQ跌1.48%,
  做空赚的是**大盘下跌的β, 不是选股alpha**。剥掉β, 这些票相对QQQ反而跑赢(做空=亏)。
- **结论**: "Sell侧有edge/Buy侧没有"很可能是同一个大盘β的两面。stockusbinance的价值前提
  (stockusbot能选出个股层面跑赢/跑输的票)**当前数据不支持** —— 收益主要来自押对大盘方向,
  那不需要昂贵的基本面AI, 做空指数即可。
- 纪律保留: n=13探索性; Sell侧OLS β(MU/MRVL高达2.3-2.7)极不稳, 精确值不可信,
  **能确定的只有方向: 纯alpha远小于+1.3%, 大概率不为正**。需扩样本至n≥30验证。

### 任务3 费率倾斜 → ⚪ null(无增量)
- 反事实: 462种随机丢5多头, 均值+0.317%; 倾斜规则+0.302% = 第47百分位 = **等于随机**。
- "改善"只是仓位往空头偏(side-mix artifact), 非更好的选股。资金费符号/窗口/因果代码正确,
  但样本太小证明不了增量edge。paper_engine_v3.py(正确资金费+倾斜开关)已建, 保留备用。

## 整合重新定位

**stockusbinance 的"方向性信号桥"路线(把stockusbot Buy/Sell当方向单做)= 当前无验证alpha。**
方向单 = β单。要救只有两条: (a)β对冲版+扩样本n≥30证明残余alpha(目前为负) (b)重新设计。

**关键亮点 — C2 为什么不受这次β打击影响:**
C2用的是**同日配对超额**(C2币做空收益 − 同日其它币做空收益), 两腿都是同日做空 →
大盘β对两腿影响相等, 在差值里**天然抵消**。C2结构上就是市场中性的。
所以杀死stockusbinance方向单的β批判, **对C2不适用** → C2仍是全项目最干净的幸存信号。

**贯穿两条线(crypto警报 + stockusbinance)的同一教训:**
散户方向性alpha = 伪装的β; 只有 相对/市场中性 结构能活。
(C2同日配对、carry本想做的方向中性——但carry死于基差; 真正活的是C2这类相对信号。)
