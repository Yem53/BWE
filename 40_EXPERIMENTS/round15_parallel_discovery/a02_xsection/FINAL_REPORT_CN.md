# a02 截面相对强弱排名 — 最终报告（中文白话）

> **一句话结论**：截面信号是**真的**（做空全市场此刻最强势的币、做多最弱的币，毛收益方向稳定、99/108 组合的多空价差为正、t 值显著），**但被交易成本结构性吃光**。
> market-neutral（多空对冲）要扣**两条腿各一次往返 = 0.30%**，而最好的毛价差盈亏平衡点只要求成本 < 0.05%–0.16%，差 2–6 倍。
> 唯一一个扣成本后还正的候选（做空 24h 最强势币 mom_24h/H4/K5）在 holdout 上 edge 确实没消失（+0.36%），**但 t 只有 1.04（35 天窗口不显著）、dev 最后一折转负、且它挑的腿 82.5% 是「入场bar急拉 >0.8%」的成交不了的妖币**——是结构性无法成交的肥尾，不是可交易 alpha。
> **诚实报负：截面排名方向无可上实盘的稳健 alpha。** 与 round13/track3（119d）的结论在 8 个月数据上完全吻合并加固。

- 生成时间：2026-05-31
- 数据窗口：K线归档 **640 币 × 1h，2025-09-28 → 2026-05-24（~8 个月 / 5735 小时）** — 比 round13 多 3 个月
- **holdout 封存线：ts ≥ 2026-04-20（最近 35 天 / 840 bars）**，dev 全程不碰、最后只读 1 次；dev = 2025-09-28 → 2026-04-20（4895 bars / ~204 天），5 折 walk-forward
- 成本：单边往返 taker **0.15%**（COST_RT，与 round8/12/13 对齐）；market-neutral = **0.30%**（两腿）；急拉成交风险 = 入场bar open→high overshoot **>0.8%** 标为「实盘成交不了」
- 共测 **108 个 (信号×K×持有) 组合 × 3 路（short_top / long_bottom / neutral）= 324 个策略实例**，外加诊断/审查/holdout 共约 **400+ 个配置/条件测试**

## 0. 与 round13/track3 的区别（没有重复造轮子）

round13/track3 已在 119d 上做过截面排名，但有三个明确缺口，本轮精确补上：

| 维度 | round13/track3 | 本轮 a02 |
|---|---|---|
| 数据 | 119d（2026-01-27 起，早期没回填） | **8 个月**（2025-09-28 起，多 3 个月、多 2 个完整熊市段） |
| 方向 | 主扫描**全是做多**；做空只在 gross 诊断顺带看 L-S spread | **short_top / long_bottom / neutral 三路系统扫描**，各扣成本、各折一致性 |
| market-neutral | **完全没做** | **核心**：dollar-neutral 多空对冲，严格按 **2× 成本** 记账 |
| 信号 | 纯收益动量 | + **波动归一动量 mom/ATR%**（scale-free，截面上不偏向高波动妖币） |
| holdout 线 | 04-24 | 本轮纪律 **04-20** |

**关键诚实点（本轮特别处理的）**：裸做空在熊市「赚钱」是 **short beta（方向性），不是 alpha**。所以 short_top 一律和「裸做空全市场基线」比 edge；真正剥离 beta 的纯截面 alpha 检验是 **market-neutral 的 gross spread**。本轮发现这 8 个月**裸做空本身就是负的**（net −0.115% ~ −0.185%），所以 short_top 的正 edge 不是 beta，是真截面信号——这反而让它更值得审，也更说明问题。

## 1. 主扫描结果（DEV，108 组合 × 3 路）

判定门槛（赢家须全中）：**net mean > 0 且 5 折 walk-forward 全部同号**；单边再要 **超裸基线 edge > 0**。

### 达标候选数

| 路 | 严格门槛（net>0 + **5/5** 折 + edge） | 放宽到 **4/5** 折 |
|---|---|---|
| **short_top**（做空最强势） | **0 / 108** | 1（仅 mom_24h H4 K5） |
| **long_bottom**（做多最弱势） | **0 / 108** | **0** |
| **market-neutral**（多空对冲） | **0 / 108** | **0** |

- **做多最弱势币反弹**：几乎全部 net 负、**0/5 折**——「跌最多的会反弹」在妖币市场不成立，与 round13 做多无 alpha 完全一致。
- **裸基线**（全程负，证明无方向 beta 可白嫖）：naive_short net H1/H2/H4 = −0.141% / −0.133% / −0.115%；naive_long 更负。

### 截面信号「存在但被吃光」的铁证

| 指标 | 数字 | 含义 |
|---|---|---|
| 多空毛价差 (neutral gross) 为正的组合 | **99 / 108** | 截面方向信息**普遍存在**且稳定 |
| 其中扣 2× 成本后 net 还正的 | **1 / 99** | 几乎全被 0.30% 吃光 |
| 最好 neutral 毛价差 | mom_24h H4 K5 **gross +0.313%**（t=3.02） | 信号显著 |
| 同组 net（扣 0.30%） | **+0.013%**（t=0.13，**不显著**，折 2/5） | 打平，经济意义为 0 |
| 多数 neutral 的**盈亏平衡成本** | **0.02% – 0.16%** | 实际要付 **0.30%** → 差 2–6 倍 |

> 这是教科书式的「signal exists but eaten by transaction cost」。在 8 个月、更大样本上重验，结论比 round13 更硬：**截面排名的多空价差真实、显著、但平均只有 0.01–0.16% 毛，而你必须付 0.30% 双边成本。**

## 2. 唯一像样的候选：做空 24h 最强势币（mom_24h / H4 / short top-K）

dev 上唯一扛过初筛的：short_top **net +0.24%（K5）/ +0.15%（K10）**，edge vs 裸做空 **+0.36% / +0.27%**，neutral gross t=3.02。long-momentum 反转做空（追高被套的镜像）。努力 KILL 它：

### 2.1 反过拟合审查（DEV）

- **5 折一致性**：K5 = **4/5**，K10/K20 = **3/5**。**最近的 fold4（2026-03-10 → 04-20，紧挨 holdout）每个 K 都转负**（−0.30% / −0.15% / −0.25%）。信号在**最接近实盘的时段衰减**——最该警惕的失败位置。
- **K 敏感性**：edge 在 K5–K10 最大、随 K 增大单调衰减到 K40 转负（+0.17→+0.24→+0.15→+0.04→−0.02）。符合「越集中在最极端的几个名字越极端」的**肥尾特征**。
- **集中度**：top5 天 = 52.8% 总盈亏、top5 币 = 34.2%（AKE/XNY/TRUTH/MYX/LAB——全是单日暴拉后崩的妖币，AKE 在 round13 也是同款幸存者）。比 round13 做多 h17（97% 来自 3 天）轻，但仍高度集中。

### 2.2 holdout reveal（读 1 次）

| 配置 | holdout net | t | 说明 |
|---|---|---|---|
| short_top mom_24h H4 K5 | **+0.171%** | **1.04** | edge vs 裸做空 = +0.356%（dev edge **确实没消失**）但 **t<2 不显著** |
| neutral mom_24h H4 K5 | +0.317%（gross +0.617%） | 1.48 | 同样方向对、同样**不显著** |
| 裸做空 H4 | −0.185% | −52.7 | 确认无 beta 白嫖 |

dev 的 edge 在 holdout 大致延续了——但 **35 天 / t=1.04 完全无法和噪声区分**，且 dev fold4 已经转负，所以「延续」很可能是 35 天窗口的运气。

### 2.3 致命点：信号挑的恰恰是**成交不了**的腿（决定性）

| 区间 | short_top K5 挑中的腿「入场bar急拉 >0.8%」占比 | 全市场同口径 |
|---|---|---|
| DEV | **82.9%** | 32.6% |
| HOLDOUT | **82.5%** | 26.2% |

做空「24h 最强势」= 系统性地去空**仍在剧烈上冲**的币，成交不了的概率是随机币的 **2.5–3 倍**。模型里的 +0.17% net 假设你能在这些 82% 会盘中再冲 >0.8% 的币上、按 open 价拿到空单——这正是实盘**拿不到**的成交。
- 印证：脚本里「弃掉 >0.8% 急拉单」后只剩 18% 的单、净值跳到 +3.17%——这**不是利好**，而是说明这条策略的「盈利」高度依赖于在急拉币上成交；真实滑点会把这 0.17% 的薄 edge 吃掉甚至吃穿。

**候选判定：REJECT。** dev 不达 5/5 折 + 紧挨 holdout 的 fold4 转负 + holdout t<2 + **结构性无法成交（82% 急拉腿）**。是「做空暴拉妖币」的肥尾，不是可交易 alpha。

## 3. 反过拟合 / 前视 / 成交性 自查

- **前视**：排名只用 close[t] 及之前；入场 = open[t+1]；持有 H 小时在 close[t+H] 平。`test_alignment.py` 用合成数据**逐笔单测核对**了 forward_return 索引对齐（hold=1/2 都对、末尾无未来 bar 的格子正确剔为 NaN）、short/long/neutral 的符号与成本记账（neutral 成本 = 单边 2×）。截面 rank 在同一时刻横截面计算，天然无前视。
- **成本诚实**：market-neutral **不是减半**（任务描述里那个猜想被证伪）——它对冲掉 beta，但两条腿各付一次往返，成本反而是单边的 **2 倍**。这正是 neutral 比单边更难赚的原因。
- **beta vs alpha**：本轮明确拆开。裸做空（short beta）这 8 个月是负的，所以 short_top 的正 edge 是真截面 alpha 而非 beta——但即便如此也跨不过成本 + 成交性。
- **多重比较**：324 个策略实例 + 诊断。任何「显著」项都额外查 5 折一致 + 集中度 + 成交性，没有单点 t 高就当真（mom_24h H4 K5 正是 dev t 高但被这三关筛掉）。
- **幸存者偏差**：唯一候选 34–53% 盈亏来自 top5 币/天，且都是暴拉妖币 → 肥尾。
- **成交性（本方向的结构性死穴）**：截面排名做空追的恰恰是「全市场此刻最猛」的币，正是急拉、滑点最大、最可能成交不了的标的（82% 入场bar >0.8% overshoot）。**这是截面排名做空无法变现的根本原因**，不是参数问题。

## 4. 最终交付结论

| 交付项 | 结论 |
|---|---|
| ① 截面排名 short/long/neutral（DEV，108×3） | **0 候选达标**（5/5 折门槛）；放宽 4/5 也仅 1 个 short，被审查 KILL |
| ② 「被成本吃光」是否成立 | **成立且量化**：99/108 组合多空毛价差为正（信号真）、盈亏平衡成本 0.02–0.16% vs 实付 0.30%，只 1/99 扣成本后勉强打平且不显著 |
| ③ market-neutral 能否靠自带对冲省成本过 holdout | **不能**。对冲掉的是 beta 不是成本；成本反而 **2×**。holdout 上唯一正的 neutral t=1.48 不显著 |
| ④ holdout 确认的可交易规则 | **无。诚实报负。** 唯一候选 holdout edge 虽延续但 t<2 + 82% 腿成交不了 → 结构性不可交易 |
| ⑤ 组合数 + 去污染 + 成本顾虑 | 324 实例 + ~400 配置；前视已单测核对；**核心顾虑 = 截面做空 = 追最猛币 = 82% 成交不了**，记录在案 |

**给项目的话**：在 8 个月（比 round13 多 3 个月、含更多熊市）上，从 short/long/market-neutral 三路系统重验了「截面相对强弱排名」方向。**截面方向信息客观存在**（做空最强、长动量反转，与项目 SHORT-only 先验一致），但 (a) 多空对冲的毛价差 0.01–0.16% 远小于 0.30% 双边成本，(b) 单边做空最强势币虽有 +0.24–0.36% edge，却**结构性地挑中 82% 成交不了的急拉妖币**。两条路都不可上实盘。
- 与 round13/track3 完全吻合：**截面排名（无论做多做空对冲）无可变现 alpha**。
- 若未来再碰：唯一可能的方向是把成本砍到 < 0.05%（maker 挂单 / 返佣 / 更长持有摊薄），或换「能在不急拉时进场的」标的——但那已不是「追全市场最猛」的截面排名思路了。本方向建议收手。
